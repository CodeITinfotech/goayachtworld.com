# Goa Yacht World

A modern, responsive website for yacht rentals in Goa, India.

## Features

- 🚤 Beautiful yacht fleet showcase
- 💬 WhatsApp integration for instant booking
- 📱 Fully responsive design
- 🔧 Admin panel for managing yachts (?admin=true)
- 💾 LocalStorage database for yacht data

## Admin Panel

Access the admin panel by:
1. Adding `?admin=true` to the URL, OR
2. Pressing `Ctrl+Shift+A` on the page

The admin panel allows you to:
- View and manage yacht fleet
- Add new yachts with images
- Edit existing yacht details (name, description, price, capacity)
- Configure WhatsApp number and booking settings
- Upload up to 10 images per yacht

## Deployment

This site is deployed on GitHub Pages at: https://codeitinfotech.github.io/goayachtworld

### For GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select "main" branch as source
4. Site will be available at `https://yourusername.github.io/repo-name`

## Local Development

Simply open `index.html` in a web browser. No build process required!

## Tech Stack

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- LocalStorage for data persistence
